# 案例任务（可复制粘贴）

挑一个场景直接跑。每个页面都尽量做成 **复制粘贴即可复现** 的形式。

<div class="grid cards" markdown>

- :material-book-open-variant: **新仓库上手（10 分钟）**  
  为刚 clone 的代码库生成一份快速上手笔记。  
  [打开](repo-onboarding.md)

- :material-magnify: **网页资料调研 Memo**  
  用 `web_search` + `web_fetch` 写一份带来源链接的简短备忘录。  
  [打开](web-research-memo.md)

- :material-web: **浏览器取证（截图 + 关键信息）**  
  打开真实网页，抽取关键字段，并保存截图作为证据。  
  [打开](browser-evidence.md)

- :material-file-pdf-box: **PDF 总结**  
  用 `pdf` 提取 PDF 文本并生成简短总结。  
  [打开](pdf-summarize.md)

- :material-stethoscope: **配置健康检查**  
  运行 `config validate` + `doctor`，生成可执行修复清单。  
  [打开](config-health-check.md)

- :material-test-tube: **修复一个失败测试**  
  只选一个失败测试，做最小安全修复并复测。  
  [打开](fix-one-failing-test.md)

- :material-clipboard-check: **发布就绪审计**  
  在打 tag 前检查版本、changelog、workflow 和打包准备。  
  [打开](release-readiness-audit.md)

- :material-alert-outline: **最近提交风险评审**  
  审查最近改动并输出按严重级别排序的风险列表。  
  [打开](commit-diff-risk-review.md)

- :material-source-branch: **Provider 路由方案**  
  产出本地/团队/CI 三套路由与成本-时延-质量权衡。  
  [打开](provider-routing-plan.md)

- :material-radar: **回归测试缺口地图**  
  找出高风险缺口并给出优先补测清单。  
  [打开](regression-test-gap.md)

- :material-rocket-launch: **10 个可复制任务**  
  一套可直接运行的模板，覆盖上手、编码、评审、文档和发布准备。  
  [打开](ten-copy-paste-tasks.md)

</div>
